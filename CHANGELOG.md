# Changelog

## v1.4.0 - 2026-06-20

### Added

- 延迟战报支持将负延迟数据识别并显示为“超时”
- 延迟战报增加分段耗时日志，便于定位面板、渲染和 Telegram 上传延迟
- 群聊统计信息与服务器详情支持直接发送最终内容

### Changed

- 延迟战报按节点并发取数，显著缩短图片生成时间
- 内联查询移至后台线程，连续输入时不再阻塞主轮询
- 临时图片消息改为后台删除，避免删除请求阻塞战报发布
- 群聊成功生成的延迟战报不再显示刷新按钮
- 机器人用户名改为运行时动态生成，不再依赖固定用户名

### Security

- 默认图片服务地址改为 `127.0.0.1`
- 示例面板统一使用 `komari.example.com`
- 发布内容不包含 Bot Token、Komari API Key、面板域名、VPS IP 或数据库
